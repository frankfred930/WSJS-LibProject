# Finnish Live Train Tracker 


## Työn jakautuminen 
Projektin toteutin itsenäisesti.

- UI/UX-suunnittelu: moderni tumma teema, responsiivinen layout
- Dataintegraatio: Digitraffic API:n hakujen toteutus Axiosilla
- Reaaliaikainen kartta: Leaflet + junien live-sijainnit
- DOM-käsittely: jQueryllä dropdownit, hakutoiminnot ja sisältöjen päivitys

## Tietoja sovelluksesta
Finnish Live Train Tracker on selaimessa toimiva sovellus, joka näyttää:

- Reaaliaikaiset junien sijainnit Suomessa
- Valitun aseman aikataulut Digitrafficin datasta
- Interaktiivisen karttanäkymän Leafletillä

## Teknologiat
- HTML5
- CSS3 + Bootstrap 5
- JavaScript
- jQuery
- Axios
- Leaflet.js
- Digitraffic API
